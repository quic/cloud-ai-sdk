## Description

This directory contains image segmentation models for Qualcomm Cloud AI 100 accelerators. Image segmentation models partition an image into distinct regions or objects, enabling pixel-level understanding.

## Models

### Mask R-CNN

Mask R-CNN is a powerful instance segmentation model that detects objects and generates pixel-perfect masks for each instance. See the [Mask R-CNN](mask_rcnn/) subdirectory for:

- Detectron2-based implementation
- Model export and compilation scripts
- Triton server deployment
- Inference client examples
